# Module 3 Checkpoint Quiz: AI-Robot Brain (NVIDIA Isaac)

This short quiz will test your understanding of AI perception, VSLAM, and navigation concepts using NVIDIA Isaac, as covered in Module 3. Choose the best answer for each question.

---

**1. What is the primary purpose of NVIDIA Isaac Sim in the robotics development workflow?**
    a) To control physical robots in real-time.
    b) To provide GPU-accelerated, photorealistic simulation for AI training and development.
    c) To generate ROS 2 launch files automatically.
    d) To manage hardware dependencies for NVIDIA GPUs.

**2. Which AI robotics concept involves building a map of an unknown environment while simultaneously estimating the robot's position within that map, using visual data?**
    a) Path Planning
    b) Object Detection
    c) Semantic Segmentation
    d) Visual SLAM (VSLAM)

**3. What is Nav2 primarily used for in ROS 2?**
    a) Developing custom robot hardware.
    b) Providing a framework for autonomous navigation (localization, path planning, control).
    c) Simulating robot physics in Gazebo.
    d) Generating synthetic sensor data.

**4. When configuring Nav2 for a bipedal robot, which component requires specialized tuning or a custom plugin to handle stable gait and balance?**
    a) `map_server`
    b) `amcl`
    c) `controller_server`
    d) `bt_navigator`

**5. What is a key advantage of using Isaac ROS packages for robotic applications?**
    a) They eliminate the need for ROS 2.
    b) They provide hardware-accelerated ROS 2 functionalities optimized for NVIDIA GPUs.
    c) They are compatible only with non-NVIDIA hardware.
    d) They are primarily used for web-based robot control interfaces.

---

## Answers

1.  **b) To provide GPU-accelerated, photorealistic simulation for AI training and development.**
2.  **d) Visual SLAM (VSLAM)**
3.  **b) Providing a framework for autonomous navigation (localization, path planning, control).**
4.  **c) `controller_server`**
5.  **b) They provide hardware-accelerated ROS 2 functionalities optimized for NVIDIA GPUs.**
