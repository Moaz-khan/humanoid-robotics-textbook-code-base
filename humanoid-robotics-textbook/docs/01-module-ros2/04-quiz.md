# Module 1 Checkpoint Quiz: ROS 2 Fundamentals

This short quiz will test your understanding of the core concepts of ROS 2 covered in Module 1. Choose the best answer for each question.

---

**1. What is the primary communication mechanism used by ROS 2 nodes for asynchronous, one-way data transfer?**
    a) Services
    b) Actions
    c) Topics
    d) Parameters

**2. In ROS 2, what is the purpose of a "node"?**
    a) To manage the entire ROS 2 system.
    b) An executable process that performs a single, modular computation.
    c) A file containing global configuration settings.
    d) A graphical visualization tool.

**3. Which of the following best describes a ROS 2 "Service"?**
    a) A stream of data published continuously by a sensor.
    b) A mechanism for long-running, interruptible tasks with feedback.
    c) A request/reply communication model for one-time transactions.
    d) A way to store dynamic configuration values.

**4. What information is typically described in a URDF file?**
    a) ROS 2 package dependencies and build instructions.
    b) The kinematic, dynamic, visual, and collision properties of a robot.
    c) Python scripts for ROS 2 nodes.
    d) Network configurations for inter-robot communication.

**5. You've made changes to a ROS 2 Python package. What command should you run from your workspace root (`~/ros2_ws/`) to build it before running the nodes?**
    a) `ros2 run`
    b) `source install/setup.bash`
    c) `colcon build`
    d) `pip install`

---

## Answers

1.  **c) Topics**
2.  **b) An executable process that performs a single, modular computation.**
3.  **c) A request/reply communication model for one-time transactions.**
4.  **b) The kinematic, dynamic, visual, and collision properties of a robot.**
5.  **c) `colcon build`**
